# Cursor Hijacker

# Introduction 
A Macos Application which can take control of your cursor without any permission needed. I know this isn't a vunrability, and was put in place for games so they could do stuff like first person, but I just wanted to demonstrate how this could be used badly.

# What it does
Brings you to your desktop I think before randomly moving your cursor around

# Installation
1. Download Application
2. Open it
3. Go into settings -> Privacy -> Scroll to bottem
4. Click "Allow Anyway"
5. Enter password
6. Enjoy
